#  Customer Churn Prediction - End-to-End ML Project

##  Project Overview
This project focuses on predicting customer churn using Machine Learning.  
The goal is to identify customers who are likely to leave a service, helping businesses take proactive retention actions.

---

##  Objectives
- Analyze customer data
- Identify churn patterns
- Build predictive ML models
- Deploy an interactive app for predictions

---

##  Machine Learning Pipeline

1. **Data Collection**
   - Load dataset (CSV file)

2. **Data Cleaning**
   - Handle missing values
   - Remove duplicates
   - Fix data types

3. **Exploratory Data Analysis (EDA)**
   - Data visualization
   - Correlation analysis
   - Pattern discovery

4. **Feature Engineering**
   - Encode categorical variables
   - Feature scaling
   - Create new features

5. **Modeling & Evaluation**
   - Train/Test split
   - Models used:
     - Logistic Regression
     - Random Forest
   - Evaluation metrics:
     - Accuracy
     - Precision
     - Recall
     - F1-score

6. **Prediction**
   - Predict customer churn (Yes / No)

7. **Deployment**
   - Interactive web app using **Streamlit**

---
